# Zenith➕ - AI-Powered Insurance for Gig Workers

## 🚀 Features

- **Strong Fraud Detection** - AI-powered fraud detection with device fingerprinting, IP analysis, and risk scoring
- **Dynamic Premium Calculation** - Location and weather-based premium adjustments
- **Zero-Touch Claims** - Automated claim processing with 5 weather triggers
- **Live Location Tracking** - Real-time location tracking with Google Maps
- **Weather Integration** - Live weather data and safety tips
- **KYC Verification** - Aadhar and PAN verification

## 📋 Prerequisites

- Node.js 18+
- MongoDB
- Redis (optional)
- Google Maps API Key
- OpenWeather API Key
